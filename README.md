# wireshark-traffic-analysis
## Objective
Captured and analyzed network traffic using Wireshark, focusing on DNS and TLS traffic.

## Tools Used
- Wireshark

## What I Did 
- Captured live network traffic
- Generated traffic by visiting websites
- Filtered traffic using:
 - DNS
 - Tls

## Key Findings
- Observed DNS queries resolving domain names like amazon.com
- Identified encrypted traffic using TLS protocol
- Understood how devices communicate over the network
